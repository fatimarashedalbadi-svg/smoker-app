 Smoker Prediction Web App

What This Project Does
This project is a machine learning web application that predicts whether a person is a **smoker or non-smoker** based on their personal and medical information.

The user enters data such as age, BMI, and region, and the system returns a prediction instantly.


 Model Description
The model was trained using the following features:
- Age
- Sex
- BMI (Body Mass Index)
- Number of children
- Region
- Insurance charges

Output:
- Smoker
- Non-Smoker

The trained files used:
- `smoker_model.pkl`
- `scaler.pkl`
